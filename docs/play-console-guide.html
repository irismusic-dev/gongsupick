3. 내부 테스트 트랙 업로드

3첫 빌드 업로드

좌측 메뉴: 테스트 → 내부 테스트 → 새 버전 만들기

  1. Play 앱 서명은 활성화 권장 (자동, Google이 마스터 키 보관 → 키스토어 분실 대비)
  2. App Bundle 업로드: 
    build/app/outputs/bundle/release/app-release.aab
    파일을 화면에 끌어다 놓기
  3. 버전명: 1.0.0 (1) 자동 채워짐
  4. 출시 노트(한국어): 
    첫 출시
- 공수 캘린더 입력 (0.5/1.0/1.5/2.0/휴 빠른 버튼)
- 월간/연간 정산 + CSV 내보내기
- 일정 관리 (오늘/다가오는/지난)
- 일감 정보 (네이버 카페 검색 + 채용사이트 바로가기)
- 데이터 백업/복구 (JSON)
  5. 저장 → 버전 검토 → 내부 테스트로 출시

검토 시간: 보통 수 분 ~ 수 시간.

자세한 설명 (복사해서 붙여넣기)

공수픽 (GongsuPick)

공수픽은 현장 작업자, 계장공, 생산직, 일용직을 위한
공수 계산 및 일감 정보 앱입니다.

복잡한 공수 계산부터 현장 일감 정보까지
현장에서 필요한 기능을 쉽고 빠르게 이용할 수 있습니다.

■ 주요 기능
• 공수 캘린더 — 0.5/1.0/1.5/2.0/휴 빠른 버튼으로 일별 공수 입력
• 자동 정산 — 단가 × 공수 자동 계산, 월간/연간 합계
• 세율 조정 — 소득세·지방세 등 세율 룰 적용 (참고용)
• 일정 관리 — 오늘/다가오는/지난 일정 한눈에
• 일감 정보 — 네이버 카페 검색 + 워크넷·알바천국·잡코리아 바로가기
• 데이터 백업/복구 — JSON 파일로 영구 보관
• 인앱 결제로 광고 제거 가능

■ 이런 분들에게 추천
- 계장공·생산직·일용직 등 현장 작업자
- 매일 공수와 단가를 따로 메모하고 계산하기 번거로운 분
- 출근 기록과 정산 내역을 깔끔하게 관리하고 싶은 분
- 일감 정보를 여러 사이트 돌아다니며 보기 귀찮은 분

■ 데이터 보호
모든 사용자 입력 데이터(공수, 일정, 그룹, 세율)는
사용자 기기에만 저장됩니다. 서버로 전송되지 않습니다.
