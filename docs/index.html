누구를 위한 앱인가요?

  • 매일 공수를 기록하고 월말마다 정산해야 하는 현장 작업자
  • 여러 거래처·현장을 동시에 관리하는 계장공·생산직
  • 단가 변동이 잦아 자동 계산이 필요한 일용직 종사자

서버 없이 기기 내부에만 데이터를 저장합니다. 회원가입도, 클라우드 동기화도 없습니다. 필요할 때 JSON 백업 파일을 직접 만들고 보관하실 수 있습니다.
