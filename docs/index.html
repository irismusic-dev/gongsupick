1. 적용범위

본 방침은 모바일 애플리케이션 공수픽 (GongsuPick)(이하 "본 앱")에 적용됩니다. 본 앱은 사용자가 입력한 모든 공수·일정·세율 데이터를 사용자의 기기 내부에만 저장하며, 운영자(이하 "개발자")가 운영하는 서버로 사용자 입력 데이터를 전송하지 않습니다.

2. 본 앱이 직접 수집·전송하는 정보

없음. 본 앱은 회원가입, 로그인, 클라우드 동기화 기능을 제공하지 않으며, 사용자가 입력한 공수·일감·일정 데이터는 사용자의 기기 외부로 전송되지 않습니다.

3. 제3자 SDK 및 외부 서비스

본 앱은 아래의 제3자 서비스를 통해 일부 기능을 제공합니다. 각 서비스는 자체 개인정보처리방침에 따라 정보를 수집할 수 있습니다.

서비스목적수집 항목방침 링크
Google AdMob 모바일 배너 광고 게재 광고 식별자(AAID/IDFA), 일반 IP, 광고 노출·클릭 데이터, 기기 정보 정책
Google Firebase Analytics 앱 사용 통계 (앱 시작, 화면 진입 등 5종 핵심 이벤트) 앱 인스턴스 ID, 화면 이름, 이벤트명·파라미터(공수 입력 여부 등 비식별 정보) 정책
Google Play Billing / Apple App Store 광고 제거 인앱 결제(선택) 결제 영수증, 사용자 결제 식별자(스토어가 발급) Play · App Store
Naver Search OpenAPI선택 네이버 카페 일감 글 검색 API 인증 키만 사용. 사용자 식별 정보를 전송하지 않습니다. 정책

본 앱은 위 서비스 외에 사용자의 개인정보를 수집·이용·제3자 제공하지 않습니다.

4. 데이터 저장 위치 및 보유 기간

  • 공수·일정·세율·그룹 데이터: 사용자 기기 내 SQLite 데이터베이스(앱 전용 영역).
  • 설정 값: 사용자 기기 내 SharedPreferences(앱 전용 영역).
  • 일감 캐시(제목·링크): 30일 또는 200건 한도 자동 정리, 사용자 기기 내 저장.
  • 보유 기간: 사용자가 "데이터 전체 삭제" 메뉴를 사용하거나, 앱을 삭제하면 기기 내 모든 데이터가 즉시 영구 삭제됩니다.
  • 백업 파일: 사용자가 "데이터 백업/복구" 메뉴에서 직접 생성한 JSON 파일은 사용자가 선택한 위치(메일·드라이브 등)에 저장되며, 본 앱 또는 개발자는 해당 파일에 접근하지 않습니다.

5. 사용자 권리

  • 광고 식별자는 OS 설정에서 언제든지 재설정 또는 추적 거부할 수 있습니다.
    • Android: 설정 → Google → 광고 → 광고 ID 삭제
    • iOS: 설정 → 개인 정보 보호 및 보안 → Tracking → 앱이 추적을 요청하도록 허용 끄기
  • 본 앱이 보여주는 광고 동의(GDPR/UMP)는 본 앱 첫 실행 시 표시되며, 언제든지 OS 설정 또는 본 앱의 광고 동의 재설정 메뉴를 통해 변경할 수 있습니다.
  • iOS 14.5+ 사용자는 "App Tracking Transparency" 다이얼로그에서 추적 허용 여부를 직접 선택할 수 있으며, 거부 시 비개인화 광고가 표시됩니다.
  • 앱 삭제 또는 "데이터 전체 삭제" 메뉴 사용 시 기기 내 모든 사용자 입력 데이터가 즉시 삭제됩니다.
