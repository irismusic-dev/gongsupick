1. 적용범위

본 방침은 모바일 애플리케이션 공수픽 (GongsuPick)(이하 "본 앱")에 적용됩니다. 본 앱은 사용자가 입력한 모든 공수·일정·세율 데이터를 사용자의 기기 내부에만 저장하며, 운영자(이하 "개발자")가 운영하는 서버로 사용자 입력 데이터를 전송하지 않습니다.

2. 본 앱이 직접 수집·전송하는 정보

본 앱은 회원가입·로그인·클라우드 동기화 기능을 제공하지 않습니다. 사용자가 입력한 공수·일감·일정·세율·그룹 데이터는 모두 사용자의 기기에만 저장되며, 개발자가 운영하는 서버로 전송되지 않습니다.

다만 아래 제3자 SDK(섹션 3)를 통해 일부 비식별 정보가 자동 수집·전송됩니다. 모든 통신은 HTTPS(TLS 1.2 이상)로 암호화됩니다.

3. 제3자 SDK 및 외부 서비스

본 앱은 아래의 제3자 서비스를 통해 일부 기능을 제공합니다. 각 서비스는 자체 개인정보처리방침에 따라 정보를 수집할 수 있습니다.

4. 데이터 저장 위치 및 보유 기간

  • 공수·일정·세율·그룹 데이터: 사용자 기기 내 SQLite 데이터베이스(앱 전용 영역).
  • 설정 값: 사용자 기기 내 SharedPreferences(앱 전용 영역).
  • 일감 캐시(제목·링크): 30일 또는 200건 한도 자동 정리, 사용자 기기 내 저장.
  • 보유 기간: 사용자가 "데이터 전체 삭제" 메뉴를 사용하거나, 앱을 삭제하면 기기 내 모든 데이터가 즉시 영구 삭제됩니다.
  • 백업 파일: 사용자가 "데이터 백업/복구" 메뉴에서 직접 생성한 JSON 파일은 사용자가 선택한 위치(메일·드라이브 등)에 저장되며, 본 앱 또는 개발자는 해당 파일에 접근하지 않습니다.

5. 국외 데이터 이전

위 제3자 서비스(Google · Naver) 중 Google이 제공하는 서비스(AdMob, Firebase Analytics·Crashlytics·Cloud Messaging)는 미국을 비롯한 해외 데이터 센터에서 정보를 처리·저장합니다. 본 앱을 사용함으로써 사용자는 해당 정보가 국외로 이전되어 처리될 수 있다는 점에 동의하는 것으로 간주됩니다. 데이터는 모두 HTTPS로 암호화 전송되며, 각 서비스 제공자의 보호 정책 및 표준 계약 조항(Standard Contractual Clauses)에 따라 보호됩니다.

  • 이전 항목: 섹션 3의 표에 명시된 비식별 정보 일체
  • 이전 국가: 미국(주 처리 국가) 등 Google·Naver의 해외 데이터 센터
  • 이전 시기: 사용자가 본 앱을 실행하는 동안 자동으로 발생
  • 이전 받는 자: Google LLC, Naver Corporation
  • 이전 받는 자의 보유·이용 기간: 각 서비스 제공자의 정책에 따름(통상 14개월~26개월)

6. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 주요 사용자로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다. 본 앱은 Google Play의 콘텐츠 등급 기준상 일반 사용자(전체 이용가)를 대상으로 하나, 아동의 직접 사용을 가정한 추가 보호조치(연령 인증, 부모 동의 등)는 적용하지 않습니다. 만 14세 미만 아동이 본 앱을 사용한 사실을 보호자가 인지한 경우, 즉시 앱 삭제 또는 "데이터 전체 삭제" 메뉴 사용을 권장하며, 위 문의처로 연락 시 신속히 조치합니다.

7. 사용자 권리

  • 광고 식별자 재설정·추적 거부는 OS 설정에서 언제든지 가능합니다.
    • Android: 설정 → Google → 광고 → 광고 ID 삭제
    • iOS 14.5+: 설정 → 개인 정보 보호 및 보안 → Tracking → 앱이 추적을 요청하도록 허용 끄기
    거부 시 비개인화 광고가 표시되며, 본 앱의 사용에는 제한이 없습니다.
  • iOS App Tracking Transparency — iOS 14.5 이상 디바이스에서는 본 앱 첫 실행 시 추적 허용 여부를 묻는 시스템 다이얼로그가 표시되며, 사용자는 자유롭게 거부할 수 있습니다.
  • 푸시 알림 차단 — 운영자 공지 알림은 OS 설정에서 차단할 수 있습니다.
    • Android: 설정 → 앱 → 공수픽 → 알림 → 허용 끄기
    • iOS: 설정 → 알림 → 공수픽 → 알림 허용 끄기
  • 분석·크래시 보고 비활성화 — Firebase Analytics와 Crashlytics는 본 앱의 표준 옵션으로 활성화되어 있습니다. 비활성화를 원하시면 위 문의처로 요청 시 차기 업데이트에서 사용자별 옵트아웃 메뉴를 제공해드립니다.
  • 데이터 열람·삭제 — 사용자가 본 앱에 입력한 모든 데이터는 기기 내부에만 있으므로, 앱의 "내보내기 (CSV)" 또는 "데이터 백업/복구" 메뉴로 자유롭게 열람·이동할 수 있습니다. 앱 삭제 또는 "데이터 전체 삭제" 메뉴 사용 시 기기 내 모든 사용자 입력 데이터가 즉시 영구 삭제됩니다.
  • 제3자 SDK가 보유한 비식별 정보의 삭제는 각 서비스 제공자의 정책에 따라 별도로 요청할 수 있습니다(Firebase 데이터 삭제 가이드 참조).
